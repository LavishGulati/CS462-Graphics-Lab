The program renders two spheres: one plastic (on right) and other textured (on
left). Shadows for both the spheres are visible on the ground. The light source
is located above the room, i.e. in +y coordinate.

PACKAGES REQUIRED:
1. OpenGL (GL and GLU)
2. GLUT

STEPS TO RUN THE PROGRAM:
$ g++ main.cpp -lGL -lGLU -lglut
$ ./a.out
